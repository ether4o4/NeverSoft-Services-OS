텍스트가 아닌 인터랙티브 화면을 만드는 오픈소스 AI 채팅 앱. MorsVitaEst는 영혼, 심장 박동, 영구 기억, 강화 학습을 갖춘 무료 ChatGPT 대안입니다.

일반 챗봇과 달리 MorsVitaEst는 네이티브 UI — 버튼, 폼, 테이블, 차트, 풀스크린 앱 — 를 채팅에서 직접 생성합니다. 계산기나 할 일 목록을 만들면 탭하고 사용할 수 있는 실제 인터페이스가 렌더링됩니다.

🧠 영혼과 기억 — 당신을 아는 챗봇
"영혼"을 작성하여 개성을 부여하세요. 대화 간 중요한 것을 기억하고 시간이 지남에 따라 지식을 강화합니다. 유용한 기억은 영구 개성으로 승격됩니다. 대화할수록 더 똑똑해집니다.

💓 심장 박동 — 항상 활성화된 에이전트
MorsVitaEst는 잠들지 않습니다. 일정에 따라 대기 작업, 이메일, 트렌드를 검토합니다. 주의가 필요할 때 알려주고, 괜찮으면 조용히 있습니다.

⚡ 24개 서비스 — 하나의 앱, 모든 모델
ChatGPT, Gemini, Claude, DeepSeek, Llama, Mistral, Grok 등에 연결. Ollama/LM Studio로 오프라인 사용 가능. 스마트 폴백과 내장 무료 등급 — API 키 불필요.

풀스크린 비주얼 앱 빌더. 탭 가능한 기능적 화면이 즉석 생성됩니다.

🔒 프라이버시 최우선, 오픈소스, 광고 없음
암호화 로컬 저장. 광고, 추적, 데이터 수집 없음. API는 직접 제공업체로. 완전 오픈소스.

🌍 어디서나 사용
Android, 태블릿, Chromebook, 데스크톱에서 사용 가능.

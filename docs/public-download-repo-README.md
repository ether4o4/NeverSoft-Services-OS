# NeverSoft Services OS — powered by MVE

A private, on-device AI assistant in a desktop-style Android launcher. This repo
hosts the **downloadable app** only — the source code is private.

## Download

**Latest preview APK (always current):**

[`MorsVitaEst-android-preview.apk`](https://github.com/ether4o4/NeverSoft-Services-OS/releases/download/android-preview-latest/MorsVitaEst-android-preview.apk)

The link is stable — the file behind it refreshes with every new build. Browse the
[Releases tab](https://github.com/ether4o4/NeverSoft-Services-OS/releases) for dated builds.

## Install

1. Download the APK on your Android device.
2. Open it; allow installing from your browser/files app if prompted.
3. It installs over a previous build without wiping your data.

> This is a debug-signed preview for sideloading, not a Play Store release.
